¿Qué es J-PRD®?

J-PRD (Jose - Interfaz de Desarrollo para el Repositorio Primitivo) Es un software que trabaja en red diseñado para la automatización de procesos y consultas a una base de datos, remplazando un gestor de data por una interfaz agradable para el usuario. Desde el momento en que inicia la instalación de la herramienta el cliente contara con un demo gratuito durante un periodo estimado de 30 días, donde los usuarios podrán administrar la herramienta conforme a sus necesidades, pasado los 30 días el programa solicitara una licencia que el cliente deberá adquirir para la reactivación de la Póliza eServices.


Servicios

• Instalación de la herramienta.
• Instalación Mysql Server & Apache Server.
• Migración de Productos, Clientes, Proveedores.
• 5 Horas iniciales de Capacitación.

Afiliaciones

• Póliza eServices (1 año).
• Mejoras Continuas.
• Inclusión de Módulos (Pre-implementación).
